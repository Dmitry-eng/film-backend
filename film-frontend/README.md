# Домашний кинотеатр — Frontend

React-приложение для домашнего онлайн-кинотеатра: каталог фильмов, поиск, фильтры, просмотр, комментарии, рейтинг, подписка на новинки.

## Стек

- **React 18** + **TypeScript**
- **Vite** — сборка и dev-сервер
- **React Router 6** — маршрутизация
- **Lucide React** — иконки

## Запуск

1. Установить зависимости:
   ```bash
   cd film-frontend
   npm install
   ```

2. Запустить бэкенд (Spring Boot на порту 8081), например:
   ```bash
   # из корня film-backend
   mvn spring-boot:run
   ```

3. Запустить фронтенд:
   ```bash
   npm run dev
   ```
   Приложение откроется на http://localhost:3000. Запросы к `/api` проксируются на http://localhost:8081.

## Сборка

```bash
npm run build
```

Статика будет в папке `dist/`. Для предпросмотра: `npm run preview`.

## Возможности

- **Главная** — баннер, блок «Главный фильм недели», новинки, призыв к подписке
- **Каталог** — список фильмов с поиском по названию, сортировкой (новинки / старые / по рейтингу), пагинацией
- **Страница фильма** — описание, рейтинг (оценка 1–10), комментарии; для новинок — требование подписки для просмотра
- **Подписка** — выбор плана (месяц/год), демо-оплата (без реального списания), сохранение статуса в localStorage
- **Вход** — демо-логин (имя + email в localStorage), для продакшена нужна реальная авторизация (JWT/OAuth)

## Настройка API

Бэкенд по умолчанию: `http://localhost:8081`. В dev-режиме Vite проксирует `/api` на этот адрес (см. `vite.config.ts`). Для другого хоста измените `server.proxy` или используйте переменную окружения и `import.meta.env`.

## Интеграция с бэкендом

- **POST /api/film/all** — список фильмов с фильтром и пагинацией
- **GET /api/film/:id** — данные фильма
- **POST /api/comment** — добавление комментария
- **POST /api/rating/vote** — оценка фильма

Категории и подписка на бэкенде не реализованы: категории и «новинки» считаются на фронте (по году выхода), подписка хранится в localStorage.
